Freeform (ver2)

Grid와 Share 하는 Freeform(ver2)을 만드는 방법을 안내합니다.
mighty-2.0.2.css 이상에서 추가되었습니다.
년

시작하기

Freeform(ver2)의 특징 및 참고할 내용

  • Semantic : 태그 변경. 항목명의 경우 html Tag를 div => label 로 변경
  • width 100% Freeform 생성(가로배열 freeform 생성)
  • Code Snippet를 사용한 구문 생성
  • html태그 내에 Inline Style (style="width: 100px")을 가급적 사용하지 않습니다. 모양이 특이한 경우 적절하게 사용하세요.
  • 복잡한 모양은 만들기 어렵습니다. 힘들게 똑같이 만드는 것 보다 만들기 쉬운 형태로 다시 디자인 하세요.

전체 Layout

.detail_box와 .ver2 를 적용하면 시작됩니다.
.detail_row는 한 행을 이룹니다. 예) table 태그의 tr
.detail_row_bb를 마지막 행에 추가하면 아래 border이 추가됩니다. 

  

  <div id="freeform" class="detail_box ver2">
    
    <div class="detail_row">
      ...
    </div>

    <div class="detail_row">
      ...
    </div>

    <div class="detail_row detail_row_bb">
      ...
    </div>

  </div>

Freeform Title 넣기

div.detail_box의 class에 has_title를 추가하고,
<label class="sub_title">회사정보</label>를 추가합니다. 

  
    <div id="freeform" class="detail_box ver2 has_title">
      <label class="sub_title">회사정보</label>

      <div class="detail_row">

        <label class="detail_label w100" for="COMP_NAME">회사명</label>
        <div class="detail_input_bg w140">
          <input id="COMP_NAME" type="text" >
        </div>

        <label class="detail_label w100" for="CEO_NAME">대표</label>
        <div class="detail_input_bg w140">
          <input id="CEO_NAME" type="text" >
        </div>

        <label class="detail_label w100" for="TEL">전화번호</label>
        <div class="detail_input_bg">
          <input id="COMP_NAME" type="text" >
        </div>

      </div>
    </div>

각 행의 구성 (.detail_row)

label.detail_label과 div.detail_input_bg의 쌍으로 한 항목을 구성합니다.
.w100은 width 100px 를 의미합니다. 10~1005까지 5의배수로 사용할 수 있습니다. 예) w10 w105 w500 w1005
5의 배수가 아닌 예를들어 .w276은 동작하지 않습니다.
label.detail_label과 div.detail_input_bg에만 width를 설정하세요. 안에 있는 input이나 select는 parent의 100% width로 동작합니다.

  

    <div class="detail_row">

      <label class="detail_label w100" for="COMP_NAME">회사명</label>
      <div class="detail_input_bg w140">
        <input id="COMP_NAME" type="text" >
      </div>

      <label class="detail_label w100" for="CEO_NAME">대표</label>
      <div class="detail_input_bg w140">
        <input id="CEO_NAME" type="text" >
      </div>

      <label class="detail_label w100" for="TEL">전화번호</label>
      <div class="detail_input_bg">
        <input id="COMP_NAME" type="text" >
      </div>

    </div>

마지막 칼럼 자동 채우기

마지막 칼럼의 div.detail_input_bg에 width를 지정하지 않으면 freeform width에 자동으로 맞춥니다.

  
    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="COMP_NAME">회사명</label>
        <div class="detail_input_bg w140">
          <input id="COMP_NAME" type="text" >
        </div>

        <label class="detail_label w100" for="CEO_NAME">대표</label>
        <div class="detail_input_bg">
          <input id="CEO_NAME" type="text" >
        </div>

      </div>
    </div>

마지막 칼럼 고정하기

마지막 칼럼의 div.detail_input_bg에 width를 고정하고, 뒤에 빈칼럼 <div class="row-closing"></div>

  

    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="COMP_NAME">회사명</label>
        <div class="detail_input_bg w100">
          <input id="COMP_NAME" type="text" >
        </div>

        <label class="detail_label w100" for="CEO_NAME">대표</label>
        <div class="detail_input_bg w100">
          <input id="CEO_NAME" type="text" >
        </div>

        <div class="row-closing"></div>

      </div>
    </div>

스타일 변경

기간이나, 단위, 검색버튼과 같은 특이한 스타일을 지정합니다.

검색버튼 추가

div.detail_input_bg의 class에 has_button을 추가하고, 안에 <button id="btn_custfind" class="search_find_img"></button>코드를 추가합니다.

  

    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="COMP_NAME">회사명</label>
        <div class="detail_input_bg has_button">
          <input id="COMP_NAME" type="text" >
          <button id="btn_custfind" class="search_find_img"></button>
        </div>

      </div>
    </div>

코드-명칭 스타일 검색버튼 추가

코드와 명칭 사이에 있는 <label></label> 을 지우고,
명칭 칼럼의 div.detail_input_bg class에 noborder를 추가하면, 왼쪽 Border가 지워집니다.

  

    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="COMP_NAME">회사명</label>
        <div class="detail_input_bg w100">
          <input id="COMP_CODE" type="text" >
        </div>
        <div class="detail_input_bg has_button">
          <input id="COMP_NAME" type="text" >
          <button id="btn_custfind" class="search_find_img"></button>
        </div>

      </div>
    </div>

기간 (시작-종료일)

~

시작칼럼과 종료칼럼 사이에 있는 <label></label> 을 지우고,
~를 넣기 위한 div.detail_input_bg 를 추가하고, class에 noborder text_center를 추가합니다. 

  

    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="FROM_DATE">근무기간</label>
        <div class="detail_input_bg w100">
          <input id="FROM_DATE" type="text" style="detail_date" >
        </div>
        <div class="detail_input_bg w20 noborder text_center">
          ~
        </div>
        <div class="detail_input_bg w100 noborder">
          <input id="TO_DATE" type="text" style="detail_date" >
        </div>

        <div class="row-closing"></div>

      </div>
    </div>

Text 추가(단위)

개월

텍스트를 넣기 위한 div.detail_input_bg 를 추가하고, class에 noborder와 width class명을 추가합니다.
이때, 주의할 점은 실제 텍스트의 width 보다 작게 잡을 경우, 전체적으로 배열이 잘 안맞을 수 있습니다. 

  

    <div id="freeform" class="detail_box ver2">
      <div class="detail_row detail_row_bb">

        <label class="detail_label w100" for="WORK_MM">근무월</label>
        <div class="detail_input_bg w150">
          <input id="WORK_MM" type="text" >
        </div>
        <div class="detail_input_bg w30 noborder">
          개월
        </div>

        <div class="row-closing"></div>

      </div>
    </div>
