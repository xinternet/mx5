Grid.js

SlickGrid를 사용하기 위해 *.Grid.js 파일에 정의하는 속성을 안내합니다.

*.Grid.js 예제

            
var columns_dg_1 = [ 
  {fieldName: 'COMPANY_CODE', width: 80,  must_input: 'Y', isKey: 'Y', visible: false, readonly: false, editable: true, sortable: true, header: {text: "시스템코드"}, styles: _Styles_textc, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SYS_ID',       width: 80,  must_input: 'Y', isKey: 'Y',visible: true, readonly: false, editable: true, sortable: true, header: {text: "시스템코드"}, styles: _Styles_textc, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SYS_NAME',     width: 100, must_input: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "시스템명"}, styles: _Styles_text, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'USE_YESNO',    width: 60,  must_input: 'N', default_value: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "사용유무"}, styles: _Styles_checkbox, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SORT_ORDER',   width: 60,  must_input: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "정렬순서"}, styles: _Styles_numberc, editor: _Editor_number, lookupdisplay: false }
];

칼럼 속성

속성 설명
fieldName (string) 필드명. 예) 'USER_NAME'
width (integer) 너비. 예) 100
must_input (String) 필수입력 여부 ( 'Y' , 'N' ). 예) 'Y'
visible (Boolean) 보이기 ( true , false ). 예) true
readonly (Boolean) 읽기전용 ( true , false). 예) true
editable (Boolean) 편집가능여부 ( true , false). 예) true
sortable (Boolean) 정렬여부 ( true , false). 예) true
header (Object) 헤더 정보. 예) {text: "회사코드"}
styles (Object) 스타일. 예) _Styles_textc
_Styles_text
Text, Left, 텍스트
_Styles_textc
Text, Center, 텍스트
_Styles_textr
Text, Right, 텍스트
_Styles_seq
Number, Center, 순번
_Styles_number
Number, Right, 숫자
_Styles_numberc
Number, Center, 숫자
_Styles_numberl
Number, Left, 숫자
_Styles_numberf1
Number, Right, 소수점 1자리
_Styles_numberf2
Number, Right, 소수점 2자리
_Styles_numberf3
Number, Right, 소수점 3자리
_Styles_numberf4
Number, Right, 소수점 4자리
_Styles_date
Date, Center, 날짜
_Styles_datemonth
Date, Center, 년월
_Styles_datetime
Date, Center, 시간
_Styles_checkbox
Checkbox, Center, 체크박스
_Styles_tree
Tree, Left, 트리
editor (Object) 편집기. 예) _Editor_text
_Editor_number
Number, Right, 숫자
_Editor_numberf1
Number, Right, 소수점 1자리
_Editor_numberf2
Number, Right, 소수점 2자리
_Editor_numberf3
Number, Right, 소수점 3자리
_Editor_numberf4
Number, Right, 소수점 4자리
_Editor_text
Text, Left, 텍스트
_Editor_multiline
Multiline Text, Left, 텍스트
_Editor_date
Date, Center, 날짜
_Editor_datemonth
Date, Center, 년월
_Editor_datetime
Date, Center, 시간
_Editor_checkbox
Checkbox, Center, 체크박스
_Editor_dropdown
Select, Left, 선택
lookupDisplay (String) GetItem에서 값 말고 명칭 읽기 ( true , false ). 예) true
button (String) 버튼 ( 'action' ). 예) "action"
alwaysShowButton (Boolean) 버튼 항상 보이기 ( true , false ). 예) true
merge (Boolean) 그룹에서 같은 칼럼 안보이게 하기 ( true , false ). 예) true
footerExpr (String) 버튼 항상 보이기 ( true , false ). 예) true
