칼럼 속성

속성 설명 예제
fieldName (String) 필드명 fieldName: 'USER_NAME'
width (Number) 너비 width: 100
must_input (String) 필수입력 여부 ( 'Y' , 'N' ) must_input: 'Y'
visible (Boolean) 보이기 ( true , false ) visible: true
readonly (Boolean) 읽기전용 ( true , false) readonly: true
editable (Boolean) 편집가능여부 ( true , false) editable: true
sortable (Boolean) 정렬여부 ( true , false) sortable: true
header (Object) 헤더 정보 header: {text: "회사코드"}
styles (Object) 스타일

_Styles_text
Text, Left, 텍스트
_Styles_textc
Text, Center, 텍스트
_Styles_textr
Text, Right, 텍스트
_Styles_seq
Number, Center, 순번
_Styles_number
Number, Right, 숫자
_Styles_numberc
Number, Center, 숫자
_Styles_numberl
Number, Left, 숫자
_Styles_numberf1
Number, Right, 소수점 1자리
_Styles_numberf2
Number, Right, 소수점 2자리
_Styles_numberf3
Number, Right, 소수점 3자리
_Styles_numberf4
Number, Right, 소수점 4자리
_Styles_date
Date, Center, 날짜
_Styles_datemonth
Date, Center, 년월
_Styles_datetime
Date, Center, 시간
_Styles_checkbox
Checkbox, Center, 체크박스
_Styles_tree
Tree, Left, 트리
 styles: _Styles_textc
editor (Object) 편집기

_Editor_number
Number, Right, 숫자
_Editor_numberf1
Number, Right, 소수점 1자리
_Editor_numberf2
Number, Right, 소수점 2자리
_Editor_numberf3
Number, Right, 소수점 3자리
_Editor_numberf4
Number, Right, 소수점 4자리
_Editor_text
Text, Left, 텍스트
_Editor_multiline
Multiline Text, Left, 텍스트
_Editor_date
Date, Center, 날짜
_Editor_datemonth
Date, Center, 년월
_Editor_datetime
Date, Center, 시간
_Editor_checkbox
Checkbox, Center, 체크박스
_Editor_dropdown
Select, Left, 선택
 editor: _Editor_text
lookupDisplay (String) GetItem에서 값 말고 명칭 읽기 ( true , false ) lookupDisplay: true
button (String) 버튼 ( 'action' ) button: "action"
alwaysShowButton (Boolean) 버튼 항상 보이기 ( true , false ) alwaysShowButton: true
headerToolTip (String) 헤더에 마우스 오버되면 설명이 표시됩니다 headerToolTip: "본사 관리팀 승인여부"
merge (Boolean) 그룹에서 같은 칼럼 안보이게 하기 ( true , false ) merge: true
maxLength (Number) 입력가능 최대자리수 maxLength: 10
footerExpr (String) Footer 표시 ( SUM , AVG , MIN , MAX , CNT , COUNT , AUTO )
SUM
Number, 합계
AVG
Number, 평균
MIN
Number, 최소값
MAX
Number, 최대값
CNT, COUNT
Number, 행의 개수
AUTO
Value, Callback 함수 리턴값
그외 텍스트
String, 텍스트 출력 예) footerExpr: "소계"
 footerExpr: "SUM"
footerAlign (String) Footer 정렬 ( left , center , right ) footerAlign: "left"
footerStyle (json) Footer custom CSS style footerStyle: {"color": "#4F4F4F"}
footerPre (String) Footer 앞에 찍힐 글자 footerPre: "총 "
footerPost (String) Footer 뒤에 찍힐 글자 footerPost: " 건"



footerExpr 속성의 AUTO 사용하기 예제

행의 값에 따라 계산이 필요한 경우, callback 함수를 정의해서 footer 표시 값을 계산합니다.

function xe_GridDataLoad2(a_dg){

  dg_1.SetAutoFooterValue("VAT_AMT", function (i) {
    var data = dg_3.GetData();
    var sum  = 0;

    for ( var i = 0, ii = data.length; i < ii; i++ ) {
      if ( data[i].CHOICE_YN == "Y" ) {
        sum += Number(data[i].A_VAT_AMT);
      }
    }

    return _X.FormatCommaPoint2( sum, 0, false);
  });

}
