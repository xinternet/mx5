Grid.js

SlickGrid를 사용하기 위해 *.Grid.js 파일에 정의하는 속성을 안내합니다.

*.Grid.js 예제


var columns_dg_1 = [ 
  {fieldName: 'COMPANY_CODE', width: 80,  must_input: 'Y', isKey: 'Y', visible: false, readonly: false, editable: true, sortable: true, header: {text: "시스템코드"}, styles: _Styles_textc, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SYS_ID',       width: 80,  must_input: 'Y', isKey: 'Y',visible: true, readonly: false, editable: true, sortable: true, header: {text: "시스템코드"}, styles: _Styles_textc, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SYS_NAME',     width: 100, must_input: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "시스템명"}, styles: _Styles_text, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'USE_YESNO',    width: 60,  must_input: 'N', default_value: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "사용유무"}, styles: _Styles_checkbox, editor: _Editor_text, lookupdisplay: false },
  {fieldName: 'SORT_ORDER',   width: 60,  must_input: 'N', visible: true, readonly: false, editable: true, sortable: true, header: {text: "정렬순서"}, styles: _Styles_numberc, editor: _Editor_number, lookupdisplay: false }
];

시작하기

bootstrap3 이 적용되고 ie8이 가능한 Theme (bs3 기반) 에 해당됩니다.

적용된 Theme

  • cop

Vertical

.x5-row 를 적용하면 세로 높이는 안에 들어있는 요소의 height에 따라 자동으로 고정됩니다.
.x5-h-(number) 를 적용한 요소는 height가 (number)px 로 고정됩니다. (number)는 5~1000까지 5px단위로 사용 가능합니다. 예) .x5-h-35 (O), .x5-h-32 (X)
.x5-h-rate-(number) 를 적용한 요소는 Parent height의 (number)% 로 height가 적용되고, window resize에 따라 같은 비율로 height가 변경됩니다. (number)는 1~100까지 사용 가능합니다. 예) .x5-h-rate-35 (O), .x5-h-rate-32 (O)
.x5-h-fill 를 적용한 요소는 parent의 남은 height를 채웁니다. window resize에 따라 높이가 자동 변경됩니다. .x5-h-fill 요소가 여러개일 경우, 남은 공간을 비율로 나눠서 채웁니다. ie8에 적용하기 위해서 .x5-h-fill는 자바스크립트로 높이를 계산합니다. 사용하지 않은 레이아웃이 더 빠르게 동작할 가능성이 있습니다. 

      
      <div class="x5-row">
        ...
      </div>

      <div class="x5-h-30">
        ...
      </div>

      <div class="x5-h-rate-30">
        ...
      </div>

      <div class="x5-h-fill">
        ...
      </div>

Horizontal

.x5-w-(number) 를 적용한 요소는 width 가 (number)px 로 고정됩니다. (number)는 5~1000까지 5px단위로 사용 가능합니다. 예) .x5-w-35 (O), .x5-w-32 (X)
.x5-w-rate-(number) 를 적용한 요소는 parent width에 (number)% 로 width가 적용되고, window resize에 따라 같은 비율로 width가 변경됩니다. (number)는 1~100까지 사용 가능합니다. 예) .x5-w-rate-35 (O), .x5-w-rate-32 (O)
.x5-w-fill 를 적용한 요소는 parent 의 남은 width를 채웁니다. .x5-w-fill 요소가 여러개일 경우, 남은 공간을 비율로 나눠서 채웁니다.(안에 내용이 들어가야 합니다.)
.x5-col-(number) 한 row를 12column으로 나눴을 때, 비율로 동작합니다. (number)는 1~12까지 정수 사용

left 고정

      </div>

        ...
      </div>

        ...
      </div>

      <div class="x5-w-300">
        ...
      </div>

비율 3등분

        ...

        ...

      </div>

FreeForm

.x5-row row를 구분합니다.
.x5-w-(number) 를 적용한 요소는 width 가 (number)px 로 고정됩니다.
.x5-w-rate-(number) 를 적용한 요소는 parent width에 (number)% 로 width가 적용되고, window resize에 따라 같은 비율로 width가 변경됩니다.
<label></label> form control의 제목일 경우, lable 태그를 사용합니다.
